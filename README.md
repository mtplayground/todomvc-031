# TodoMVC 031

TodoMVC implementation scaffolded with Vite, React, and TypeScript.

## Requirements

- Node.js 20+
- npm 10+

## Run locally

```bash
npm install
npm run dev
```

The Vite dev server runs on `0.0.0.0:8080`.

## Build for production

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```
